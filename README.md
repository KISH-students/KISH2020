# KISH Application
> 하노이한국국제학교 비공식 어플 리포지토리에 오신걸 환영합니다.

KISH APP을 통해 급식, 시험 일정과 같은 정보를 간편하게 확인할 수 있습니다!

이 리포지토리는 KISH 학생이라면 누구나 기여할 수 있습니다.
